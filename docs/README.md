# Admin Policy Documents

이 폴더는 어드민 설계정책서를 Git에서 관리하기 위한 문서 저장소입니다.

## 운영 원칙

- `admin.csv`는 메뉴 인벤토리와 초기 상태를 관리하는 기준 파일로 둔다.
- `docs/menus/{MENU_CODE}.md`는 메뉴별 상세 기획지침, 개발 기준, 권한/감사 정책을 관리하는 기준 문서로 둔다.
- 화면 개발을 시작할 때는 메뉴 코드 기준으로 해당 MD 문서를 먼저 확인한다.
- 구현이 설계와 달라지는 경우에는 해당 메뉴 MD의 개발 지침 또는 변경 이력에 사유를 남긴다.
- 여러 메뉴가 하나의 업무 흐름으로 묶이는 경우에는 `docs/domains/`에 도메인 단위 정책 문서를 추가한다.

## 메뉴 문서 위치

- 메뉴별 문서: `docs/menus/`
- 도메인 문서: `docs/domains/`

## 개발 지시 예시

`ADM-RISK-004 차단 관리대장 화면은 docs/menus/ADM-RISK-004.md 설계 기준대로 개발해.`

이 지시를 받은 개발자는 해당 문서의 화면 목적, 탭 구성, 집계 기준, 권한/감사 기준을 우선 확인하고 구현해야 합니다.
