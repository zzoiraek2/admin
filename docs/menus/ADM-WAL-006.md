# ADM-WAL-006 오입금 반환관리

## 기본 정보

- 메뉴 코드: ADM-WAL-006
- 상단 메뉴: 입출금·지갑
- 좌측 1메뉴: 입금 운영
- 좌측 2메뉴: 오입금 반환관리
- 상태: 준비중
- 구 어드민 매핑: 신규 정책 정의

## 화면 목적

오입금 반환 요청을 접수, 검증, 승인, 실행한다.

## 업무 설명

오입금 유형, 소유권 증빙, 반환 가능 여부, 반환주소, 수수료, 반환 TXID를 관리하는 처리 화면이다. 반환은 외부 전송이 발생하는 출금성 업무이므로 반환 승인과 실행은 결재 완료 후 수행한다.

## 결재 프로세스

- 반환 정보: 반환 대상, 반환주소, 수수료, 반환수량 확정은 결재 상신 전 검증 근거를 남긴다.
- 승인·실행: 오입금 반환 승인, 반환 불가 확정, 반환주소 변경, 실행 재시도는 승인 완료 후 처리한다.

## 탭 구성

- 반환 접수
- 증빙·소유권 검증
- 반환 정보
- 승인·실행
- 반환 이력

## 탭별 상세 설계

### 반환 접수

- 업무 목적: 오입금 반환 요청 또는 내부 발견 건을 접수한다.
- 검색 조건: 기간, 회원번호, 자산, 오입금유형, 접수상태
- 목록 컬럼: 접수번호, 접수일시, 회원번호, 자산, 금액, 오입금유형, TXID, 접수상태, 담당자
- 상세보기 정보: 접수 경로, 요청 내용, 관련 TX, 입금주소, 회원 정보, 초기 판단
- 주요 기능: 접수 조회, 담당자 배정, 관련 입금 이동, 증빙 검증 이동
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-WAL-004 가상자산 입금 현황, ADM-MEM-001 회원 기본조회
- 권한/감사: 접수 조회와 담당자 배정 이력을 감사로그로 남긴다.

### 증빙·소유권 검증

- 업무 목적: 반환 요청자의 소유권과 반환 가능 여부를 검증한다.
- 검색 조건: 접수번호, 회원번호, 검증상태, 증빙유형, 담당자
- 목록 컬럼: 접수번호, 회원번호, 자산, 증빙유형, 검증상태, 보완필요 여부, 담당자, 최근검토시각
- 상세보기 정보: 제출 증빙, 지갑주소 소유권 근거, TX 원장, 반환 가능/불가 사유, 검토 의견
- 주요 기능: 증빙 확인, 보완 요청, 검토 의견 등록, 반환 가능 여부 표시
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-CS-004 증빙 제출내역, ADM-WAL-004 가상자산 입금 현황
- 권한/감사: 증빙 조회, 보완 요청, 검토 의견 이력을 감사로그로 남긴다.

### 반환 정보

- 업무 목적: 반환 대상 금액, 반환주소, 수수료, 네트워크를 확정한다.
- 검색 조건: 접수번호, 회원번호, 반환상태, 자산, 네트워크
- 목록 컬럼: 접수번호, 회원번호, 자산, 반환수량, 수수료, 반환주소, 네트워크, 상신상태
- 상세보기 정보: 반환주소 검증, 반환수량 산출, 수수료 차감 기준, 반환 불가 조건, 상신 내용
- 주요 기능: 반환 정보 입력, 주소 검증, 수수료 계산, 결재 상신
- 결재/승인: 반환 대상, 반환주소, 수수료, 반환수량 확정은 결재 상신 전 검증 근거를 남긴다.
- 연계 메뉴: ADM-WAL-013 지갑/주소 관리, ADM-RISK-011 KYT Alert
- 권한/감사: 반환 정보 입력, 주소 검증, 상신 이력을 감사로그로 남긴다.

### 승인·실행

- 업무 목적: 승인된 반환 건을 실행하고 TXID와 결과를 확인한다.
- 검색 조건: 기간, 승인상태, 실행상태, 회원번호, 자산
- 목록 컬럼: 상신번호, 접수번호, 회원번호, 자산, 반환수량, 승인상태, 실행상태, 반환TXID, 처리시각
- 상세보기 정보: 승인 의견, 반환 실행 요청값, 브로드캐스트 결과, TXID, 실패/재시도 이력
- 주요 기능: 승인, 반려, 보완 요청, 반환 실행, 실행 재시도, 반환 불가 확정
- 결재/승인: 오입금 반환 승인, 반환 불가 확정, 반환주소 변경, 실행 재시도는 승인 완료 후 처리한다.
- 연계 메뉴: ADM-WAL-007 가상자산 출금 현황, ADM-SEC-007 감사로그
- 권한/감사: 승인/반려, 반환 실행, TXID, 실패 사유를 감사로그로 남긴다.

### 반환 이력

- 업무 목적: 오입금 반환 처리 결과와 감사 근거를 조회한다.
- 검색 조건: 기간, 회원번호, 자산, 처리결과, 담당자
- 목록 컬럼: 처리시각, 접수번호, 회원번호, 자산, 반환수량, 처리결과, 반환TXID, 담당자
- 상세보기 정보: 접수부터 실행까지의 처리 타임라인, 승인 이력, TX 정보, 감사로그
- 주요 기능: 반환 이력 조회, TX 확인, 감사로그 이동, 다운로드
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-SEC-007 감사로그, ADM-WAL-007 가상자산 출금 현황
- 권한/감사: 반환 이력 조회와 다운로드 이력을 감사로그로 남긴다.

## 설계 방식 제안

- 오입금 반환은 외부 전송이 발생하므로 승인과 실행을 분리한다.
- 반환주소, 반환수량, 수수료는 승인 전후 변경 이력을 모두 남긴다.
- 반환 불가 처리도 고객 영향이 있으므로 사유와 승인 이력을 남긴다.

## 개발 지침

ADM-WAL-006 화면은 오입금 반환 접수, 증빙 검증, 결재 승인, 실행 이력을 기준으로 개발한다. 승인 없는 반환 실행은 허용하지 않는다.
설계와 다르게 구현해야 하는 항목은 변경 사유와 대체 정책을 작업 기록에 남긴다.
