# ADM-WAL-007 가상자산 출금 현황

## 기본 정보

- 메뉴 코드: ADM-WAL-007
- 상단 메뉴: 입출금·지갑
- 좌측 1메뉴: 출금 운영
- 좌측 2메뉴: 가상자산 출금 현황
- 상태: 정의됨
- 구 어드민 매핑: 신규 정책 정의

## 화면 목적

고객 가상자산 출금 신청과 처리 진행 상태를 조회한다.

## 업무 설명

출금 신청, 승인/보류 상태, 브로드캐스트, TXID, 완료, 실패 상태를 조회하는 화면이다. 고위험 승인, 보류 해제, 수동 재처리, 출금 취소/대체 처리는 전용 메뉴에서 수행한다.

## 결재 프로세스

- 해당사항 없음

## 탭 구성

- 출금 요약
- 처리 진행
- 보류·지연

## 탭별 상세 설계

- 검색 조건: 기간, 회원번호, 자산, 네트워크, 출금상태, 위험표시
- 목록 컬럼: 신청시각, 회원번호, 자산, 네트워크, 출금금액, 수수료, 출금상태, TXID, 처리시간
- 상세보기 정보: 출금 신청 정보, 주소, 승인/보류 상태, 브로드캐스트 상태, 원장 반영 상태
- 주요 기능: 출금 조회, 상세 이동, 승인/보류 화면 이동, 실패관리 이동
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-WAL-008 출금 승인/보류, ADM-WAL-009 출금 재처리/실패관리, ADM-WAL-002 입출금 원장조회
- 권한/감사: 출금 조회, 주소 조회, 다운로드 이력을 감사로그로 남긴다.

- 목록 컬럼: 신청시각, 회원번호, 자산, 금액, 처리단계, 승인상태, 브로드캐스트상태, TXID, 지연시간

- 주요 기능: 처리 단계 조회, 지연 필터, 관련 처리 화면 이동
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-WAL-008 출금 승인/보류, ADM-WAL-009 출금 재처리/실패관리
- 권한/감사: 처리 진행 조회와 상세 이동 이력을 감사로그로 남긴다.

- 업무 목적: 보류 또는 장기 지연 출금 건을 확인하고 처리 메뉴로 연결한다.
- 검색 조건: 기간, 보류사유, 지연시간, 위험유형, 회원번호
- 목록 컬럼: 신청시각, 회원번호, 자산, 금액, 보류사유, 위험유형, 지연시간, 처리필요 여부
- 상세보기 정보: 보류 근거, KYT/Travel Rule 결과, 지연 원인, 처리 메뉴 링크
- 주요 기능: 보류/지연 조회, 승인/보류 메뉴 이동, 운영 메모
- 결재/승인: 해당사항 없음

- 권한/감사: 완료/실패 조회와 다운로드 이력을 감사로그로 남긴다.

## 설계 방식 제안

- 출금 현황은 조회 화면이며 승인, 보류 해제, 재처리 기능을 포함하지 않는다.
- 고위험 출금은 ADM-WAL-008에서, 실패 재처리는 ADM-WAL-009에서 처리한다.
- 외부 전송에 영향을 주는 기능은 반드시 결재 화면으로 분리한다.

## 개발 지침

ADM-WAL-007 화면은 출금 상태 조회와 처리 메뉴 연결을 기준으로 개발한다. 출금 승인, 보류 해제, 수동 재처리 기능은 포함하지 않는다.
설계와 다르게 구현해야 하는 항목은 변경 사유와 대체 정책을 작업 기록에 남긴다.
