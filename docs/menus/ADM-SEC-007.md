# ADM-SEC-007 다운로드 이력

## 기본 정보

- 메뉴 코드: ADM-SEC-007
- 상단 메뉴: 보안·관리자통제
- 좌측 1메뉴: 개인정보 접속기록
- 좌측 2메뉴: 다운로드 이력
- 상태: 준비중
- 구 어드민 매핑: 신규 정책 정의

## 화면 목적

개인정보 포함 파일 다운로드 통제

## 업무 설명

파일명, 건수, 사유, 승인자, 다운로드시간

## 탭 구성

- 접근 요약
- 조회·다운로드 상세
- 사유·승인
- 이상 패턴
- 감사로그


## 탭별 상세 설계

### 접근 요약

- 업무 목적: 다운로드 이력의 접근 요약 탭은 보안 관리자, 시스템 관리자, 감사 담당자, 승인권자가 개인정보 포함 파일 다운로드 통제 업무를 접근 요약 관점에서 판단하고 다음 처리 화면으로 자연스럽게 이동하도록 설계한다.
- 검색 조건: 관리자ID, 역할, 권한, IP, 기간, 작업유형, 승인상태, 이상징후, 접근 요약 상태/구분
- 목록 컬럼: 관리자, 역할/권한, 대상 메뉴, 작업유형, 사유, IP/기기, 승인상태, 작업시각, 접근 요약 주요 판단값
- 상세보기 정보: 권한 범위, 접근 이력, 작업 전후 값, 조회 사유, 마스킹 해제 여부, 승인/반려 이력, 접근 요약별 판단 기준, 운영자 메모, 관련 증빙
- 주요 기능: 권한 부여/회수 상신, 세션 종료, 설정 변경, 로그 다운로드, 이상 작업 확인, 알림 확인, 추이 비교, 드릴다운 조회, 담당 업무 큐 이동, 변경 전후 비교, 감사 로그 다운로드, 관련 승인 건 역추적
- 결재/승인: 조회와 단순 필터링은 결재 없이 가능하되, 상태 변경이나 데이터 추출이 발생하면 승인 흐름으로 전환한다.
- 연계 메뉴: 관리자 계정, 역할/권한, 개인정보 조회 로그, 공통 설정, 승인선 관리, 감사로그
- 권한/감사: 조회 사유, 검색 조건, 마스킹 해제, 다운로드, 등록/수정/삭제, 승인/반려, 실행 결과와 실패 사유를 관리자 감사로그로 남긴다.

- 목록, 상세, 처리 이력, 변경 이력을 기준 구조로 둔다.

- 변경성 액션은 사유 입력과 감사로그를 필수로 둔다.
- 관련 회원, 거래, 입출금, Case, 보고서 화면으로 이동할 수 있게 연결한다.
