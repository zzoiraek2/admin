# ADM-RISK-005 고객요청 동결 관리

## 탭별 상세 설계

### 준법 요약

- 업무 목적: 고객요청 동결 관리의 준법 요약 탭은 준법 담당자, FDS/KYT 담당자, 입출금 운영자, 승인권자가 고객 요청에 따른 계정·자산 동결 관리 업무를 준법 요약 관점에서 판단하고 다음 처리 화면으로 자연스럽게 이동하도록 설계한다.
- 검색 조건: 기간, 회원번호, 사유코드, Case 번호, 차단/해제 상태, 자산, VASP, 보고 기준, 준법 요약 상태/구분
- 목록 컬럼: 발생일시, 회원/대상, 사유코드, 차단범위, 건수/금액, 상태, 승인단계, 근거 문서, 준법 요약 주요 판단값
- 상세보기 정보: 탐지/등록 근거, 차단 사유, 원천 로그, 증빙, 승인 이력, 보고 산출 기준, 후속 조치, 준법 요약별 판단 기준, 운영자 메모, 관련 증빙
- 주요 기능: Case 검토, 차단 등록/해제 상신, 증빙 첨부, 보고 집계 확인, 원천 로그 추적, 알림 확인, 추이 비교, 드릴다운 조회, 담당 업무 큐 이동, 설정 초안 저장, 영향도 검증, 적용 예약, 비활성 처리, 롤백 요청
- 결재/승인: 등록, 변경, 해제, 제한 적용, 대량 처리, 외부 보고 값 확정은 결재 상신 후 승인 완료 시 실행한다.
- 연계 메뉴: 차단 등록, 차단 해제, 차단 사유코드, 차단 관리대장, KYT Alert, 회원 기본조회
- 권한/감사: 조회 사유, 검색 조건, 마스킹 해제, 다운로드, 등록/수정/삭제, 승인/반려, 실행 결과와 실패 사유를 관리자 감사로그로 남긴다.
