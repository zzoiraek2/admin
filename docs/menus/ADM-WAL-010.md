일/월 출금 한도, 자산별 한도, 회원등급별 한도, KYC 상태별 한도, 예외 한도를 관리하는 정책 화면이다. 출금 한도 변경은 고객의 출금 가능 범위에 직접 영향을 주므로 결재 후 반영한다.

## 결재 프로세스

- 등급별 한도: 회원 등급/KYC 상태별 한도 변경은 승인 완료 후 반영한다.
- 자산별 한도: 자산별 한도 변경은 승인 완료 후 반영한다.
- 예외 한도: 특정 회원 또는 조건의 예외 한도 등록/해제는 승인 완료 후 반영한다.
