- 업무 목적: 자산/네트워크별 지갑 인프라의 현재 상태를 빠르게 확인하는 조회 탭이다.

- 목록 컬럼: 자산, 네트워크, 지갑상태, 노드상태, 컨펌상태, 입금가능, 출금가능, 최근 갱신
- 상세보기 정보: 지갑 상태 요약, 입출금 가능 여부, 최근 장애/알림, 관련 노드와 입출금 정책 링크
- 주요 기능: 상태 조회, 필터 저장, 관련 입출금/노드 화면 이동, 알림 확인
- 결재/승인: 해당사항 없음
- 연계 메뉴: 입출금 조회, 지갑 관리, 노드 모니터링, 자산 대사, 알림 이력, 감사로그
- 권한/감사: 지갑 상태 조회, 검색 조건, 알림 확인, 관련 화면 이동, 로그 다운로드 사유를 감사로그로 남긴다.
