# ADM-WAL-013 지갑/주소 관리

## 기본 정보

- 메뉴 코드: ADM-WAL-013
- 상단 메뉴: 입출금·지갑
- 좌측 1메뉴: 지갑 관리
- 좌측 2메뉴: 지갑/주소 관리
- 상태: 준비중
- 구 어드민 매핑: 지갑 상태, 주소 관리 통합

## 화면 목적

가상자산별 지갑·주소 현황과 고객/회사 지갑, 웜월렛 설정 상태를 관리한다.

## 업무 설명

가상자산 티커, 메인넷 기반, 거래지원 여부, 메인넷 유지/종료 여부, 지갑 사용 여부, 고객자산/회사분 지갑, 핫/콜드월렛 구분, 웜월렛 여부, 지갑주소, 보유수량, 익스플로러 URL을 조회하고 관리한다. 조회는 결재 없이 가능하지만 지갑주소 등록/수정/비활성, 고객/회사 구분 변경, 웜월렛 설정 변경은 결재 후 반영한다.

## 결재 프로세스

- 주소 등록·변경: 지갑주소 등록, 수정, 비활성, 사용중지는 승인 완료 후 반영한다.
- 고객·회사 지갑 설정: 고객자산/회사분 지갑 구분, 지갑 용도, 사용 여부 변경은 승인 완료 후 반영한다.
- 웜월렛 설정: 웜월렛 여부와 비고/운영 기준 변경은 승인 완료 후 반영한다.

## 탭 구성

- 지갑 요약
- 주소별 현황
- 주소 등록·변경

## 설계 방식 제안

- 지갑/주소 조회와 설정 변경을 명확히 분리한다.
- 지갑주소, 지갑 용도, 고객/회사 구분, 웜월렛 여부 변경은 결재 후 반영한다.
- 잔고 대사와 고객자산 대사는 조회 화면으로 연결해 검증할 수 있게 한다.

## 개발 지침

ADM-WAL-013 화면은 지갑/주소 현황 조회와 결재 기반 설정 변경을 기준으로 개발한다. 승인 없는 지갑주소 등록/수정/비활성 또는 웜월렛 설정 변경은 허용하지 않는다.
설계와 다르게 구현해야 하는 항목은 변경 사유와 대체 정책을 작업 기록에 남긴다.
