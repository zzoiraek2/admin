# ADM-WAL-001 입출금 통합현황

## 기본 정보

- 메뉴 코드: ADM-WAL-001
- 상단 메뉴: 입출금·지갑
- 좌측 1메뉴: 입출금 통합조회
- 좌측 2메뉴: 입출금 통합현황
- 상태: 준비중
- 구 어드민 매핑: 신규 정책 정의

## 화면 목적

고객 입금과 출금의 전체 처리 상태를 운영 관점에서 통합 조회한다.

## 업무 설명

입금, 출금, 보류, 지연, 실패, 완료 상태를 한 화면에서 모니터링하는 조회 화면이다. 실제 수동 반영, 재처리, 승인, 보정은 개별 처리 메뉴에서 수행하며 이 화면은 현황 확인과 원천 화면 이동 역할로 한정한다.

## 결재 프로세스

- 해당사항 없음

## 탭 구성

- 통합 요약
- 입금 현황
- 출금 현황
- 보류·지연 모니터링
- 이상상태 이동

## 탭별 상세 설계

### 통합 요약

- 업무 목적: 입출금 전체 처리량과 미처리 위험을 빠르게 확인한다.
- 검색 조건: 기간, 자산, 네트워크, 입출금구분, 처리상태, 회원번호, 지연 여부
- 목록 컬럼: 구분, 전체건수, 완료건수, 보류건수, 실패건수, 지연건수, 총금액, 최근갱신시각
- 상세보기 정보: 상태별 집계, 자산별 집계, 지연 기준, 이상상태 링크
- 주요 기능: 통합 현황 조회, 상태별 필터, 관련 입금/출금 화면 이동, 다운로드
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-WAL-004 가상자산 입금 현황, ADM-WAL-007 가상자산 출금 현황, ADM-WAL-003 입출금 이상상태 조회
- 권한/감사: 조회 조건, 다운로드, 관련 화면 이동 이력을 감사로그로 남긴다.

### 입금 현황

- 업무 목적: 입금 건의 처리 상태를 요약 조회한다.
- 검색 조건: 기간, 회원번호, 자산, 네트워크, TXID, 입금상태
- 목록 컬럼: 입금일시, 회원번호, 자산, 네트워크, 금액, TXID, 컨펌수, 입금상태, 반영일시
- 상세보기 정보: 입금 요청/탐지 정보, 컨펌 진행, 회원 매칭, 원장 반영 상태
- 주요 기능: 입금 조회, 입금 상세 이동, 미반영 관리 이동, TX 확인
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-WAL-004 가상자산 입금 현황, ADM-WAL-005 입금 미반영/지연 관리
- 권한/감사: 입금 조회와 상세 이동 이력을 감사로그로 남긴다.

### 출금 현황

- 업무 목적: 출금 신청의 처리 상태를 요약 조회한다.
- 검색 조건: 기간, 회원번호, 자산, 네트워크, 출금상태, TXID, 위험표시
- 목록 컬럼: 신청일시, 회원번호, 자산, 네트워크, 금액, 수수료, 출금상태, TXID, 처리시간
- 상세보기 정보: 출금 신청 정보, 승인/보류 상태, 브로드캐스트 결과, 실패 사유
- 주요 기능: 출금 조회, 출금 상세 이동, 승인/보류 화면 이동, 실패관리 이동
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-WAL-007 가상자산 출금 현황, ADM-WAL-008 출금 승인/보류, ADM-WAL-009 출금 재처리/실패관리
- 권한/감사: 출금 조회와 상세 이동 이력을 감사로그로 남긴다.

- 검색 조건: 기간, 입출금구분, 자산, 지연시간, 보류사유, 실패사유
- 목록 컬럼: 발생시각, 구분, 회원번호, 자산, 금액, 상태, 사유, 지연시간, 처리메뉴

- 주요 기능: 지연 건 필터링, 처리 메뉴 이동, 운영 메모 등록
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-WAL-003 입출금 이상상태 조회, ADM-WAL-005 입금 미반영/지연 관리, ADM-WAL-008 출금 승인/보류

## 설계 방식 제안

- 통합현황은 조회와 관제 화면으로 설계하고 직접 처리 버튼을 두지 않는다.
- 상태 변경, 수동 반영, 출금 승인, 재처리는 각 전용 메뉴에서 수행한다.
- 다운로드는 별도 탭이 아니라 권한 기반 목록 기능으로 제공한다.

## 개발 지침

ADM-WAL-001 화면은 입출금 현황 조회와 관련 업무 이동을 기준으로 개발한다. 고객 잔고나 외부 전송에 영향을 주는 액션은 포함하지 않는다.
설계와 다르게 구현해야 하는 항목은 변경 사유와 대체 정책을 작업 기록에 남긴다.
