# ADM-WAL-001 입출금 통합현황

## 기본 정보

- 메뉴 코드: ADM-WAL-001
- 상단 메뉴: 입출금·지갑
- 좌측 1메뉴: 입출금 통합조회
- 좌측 2메뉴: 입출금 통합현황
- 상태: 준비중
- 구 어드민 매핑: 모니터링 모니터링 > 거래 현황 > 입출금 내역 

## 화면 목적

## 업무 설명

## 탭 구성

- 입출금 요약
- 처리 상태별 현황

- 수동 처리·보정
- 이력·감사


## 탭별 상세 설계

### 입출금 요약

- 업무 목적: 입출금 통합현황의 입출금 요약 탭은 입출금 운영자, 지갑 운영자, 정산/회계 담당자, 준법 담당자가 고객 입출금 상태를 운영 관점에서 조회 업무를 입출금 요약 관점에서 판단하고 다음 처리 화면으로 자연스럽게 이동하도록 설계한다.
- 검색 조건: 기간, 회원번호, 자산, 네트워크, TXID, 주소, 처리상태, 보류/실패 사유, 입출금 요약 상태/구분
- 목록 컬럼: 요청일시, 회원번호, 자산/네트워크, 금액, 수수료, 상태, TXID/주소, 위험표시, 담당자, 입출금 요약 주요 판단값
- 상세보기 정보: 원장 반영 상태, 온체인 상태, 주소/메모, 위험 탐지 결과, 처리 로그, 수동 처리 근거, 입출금 요약별 판단 기준, 운영자 메모, 관련 증빙
- 주요 기능: 상세 조회, 보류/재처리 요청, 수동 확인, 차단 연계, 지갑 작업 요청, 증빙 첨부, 알림 확인, 추이 비교, 드릴다운 조회, 담당 업무 큐 이동
- 결재/승인: 조회와 단순 필터링은 결재 없이 가능하되, 상태 변경이나 데이터 추출이 발생하면 승인 흐름으로 전환한다.
- 연계 메뉴: 입출금 조회, 지갑 관리, 자산 대사, 차단 등록, KYT Alert, 정산/회계
- 권한/감사: 조회 사유, 검색 조건, 마스킹 해제, 다운로드, 등록/수정/삭제, 승인/반려, 실행 결과와 실패 사유를 관리자 감사로그로 남긴다.
