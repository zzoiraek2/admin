- 목록 컬럼: 접수/게시일시, 대상, 카테고리, 상태, 담당자, SLA, 최근 변경, 노출/처리 결과, 콘텐츠 요약 주요 판단값
- 상세보기 정보: 문의/콘텐츠 본문, 회원 맥락, 첨부/증빙, 처리 메모, 게시 조건, 변경 이력, 콘텐츠 요약별 판단 기준, 운영자 메모, 관련 증빙
- 주요 기능: 답변/처리, 이관, 보류, 게시 예약, 종료/회수, 관련 회원/거래 이동, 알림 확인, 추이 비교, 드릴다운 조회, 담당 업무 큐 이동
