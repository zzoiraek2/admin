# ADM-CS-003 상담 이관·처리내역

- 상단 메뉴: 고객지원·서비스

- 업무 목적: 상담 이관·처리내역의 문의 요약 탭은 CS 운영자, 서비스 운영자, 콘텐츠 담당자, 승인권자가 문의를 내부 부서로 이관·추적 업무를 문의 요약 관점에서 판단하고 다음 처리 화면으로 자연스럽게 이동하도록 설계한다.

- 상세보기 정보: 문의/콘텐츠 본문, 회원 맥락, 첨부/증빙, 처리 메모, 게시 조건, 변경 이력, 문의 요약별 판단 기준, 운영자 메모, 관련 증빙
- 주요 기능: 답변/처리, 이관, 보류, 게시 예약, 종료/회수, 관련 회원/거래 이동, 알림 확인, 추이 비교, 드릴다운 조회, 담당 업무 큐 이동
