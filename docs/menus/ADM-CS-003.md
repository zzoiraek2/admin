# ADM-CS-003 상담 이관·처리내역

## 기본 정보

- 메뉴 코드: ADM-CS-003
- 상단 메뉴: 고객지원·서비스
- 좌측 1메뉴: 문의/티켓 관리
- 좌측 2메뉴: 상담 이관·처리내역
- 상태: 준비중
- 구 어드민 매핑: 신규 정책 정의

## 화면 목적

문의를 내부 부서로 이관·추적

## 업무 설명

이관부서, 담당자, 처리의견, 완료여부, 재이관


## 결재 프로세스

- 처리 결과: 승인 결과와 다른 조치를 수행해야 하면 재상신 또는 예외 승인으로 처리한다.

## 탭 구성

- 문의 요약

- 답변·보류
- 처리 결과
- 사후·이력

- 목록, 상세, 처리 이력, 변경 이력을 기준 구조로 둔다.
