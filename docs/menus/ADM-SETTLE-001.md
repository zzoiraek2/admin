## 탭별 상세 설계

### 정산 요약

- 업무 목적: 매출/수수료의 정산 요약 탭은 정산 담당자, 회계 담당자, 감사 담당자, 승인권자가 거래소 매출과 수수료 수익 확인 업무를 정산 요약 관점에서 판단하고 다음 처리 화면으로 자연스럽게 이동하도록 설계한다.
- 검색 조건: 기준일, 정산구분, 자산/통화, 계정과목, 마감상태, 차이 여부, 담당자, 정산 요약 상태/구분
- 목록 컬럼: 기준일, 정산구분, 금액, 원장 금액, 차이 금액, 상태, 조정 사유, 승인상태, 정산 요약 주요 판단값
- 상세보기 정보: 산출식, 원천 데이터, 대사 결과, 차이 원인, 조정 전후 값, 승인/마감 이력, 정산 요약별 판단 기준, 운영자 메모, 관련 증빙
- 주요 기능: 집계 재생성, 차이 확인, 조정 요청, 마감 상신, 보고서 다운로드, 알림 확인, 추이 비교, 드릴다운 조회, 담당 업무 큐 이동
- 결재/승인: 조회와 단순 필터링은 결재 없이 가능하되, 상태 변경이나 데이터 추출이 발생하면 승인 흐름으로 전환한다.
- 연계 메뉴: 정산 대시보드, 자산 대사, 입출금 조회, 회계 마감, 보고서 센터, 감사로그
- 권한/감사: 조회 사유, 검색 조건, 마스킹 해제, 다운로드, 등록/수정/삭제, 승인/반려, 실행 결과와 실패 사유를 관리자 감사로그로 남긴다.
