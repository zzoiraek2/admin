## 탭별 상세 설계

### 심사 요약

- 업무 목적: 유지심사의 심사 요약 탭은 거래지원 심사 담당자, 상품 운영자, 준법 담당자, 승인권자가 기존 거래지원 자산의 유지 여부 심사 업무를 심사 요약 관점에서 판단하고 다음 처리 화면으로 자연스럽게 이동하도록 설계한다.
- 검색 조건: 프로젝트명, 자산명, 심사상태, 담당자, 위험등급, 요청일, 의결 결과, 심사 요약 상태/구분
- 목록 컬럼: 프로젝트/자산, 단계, 담당자, 위험등급, 요청일, 마감일, 의결 결과, 후속 조치, 심사 요약 주요 판단값
- 상세보기 정보: 심사 자료, 체크리스트, 리스크 검토, 의결 근거, 공지/거래상태, 변경 이력, 심사 요약별 판단 기준, 운영자 메모, 관련 증빙
- 주요 기능: 자료 요청, 보완 요청, 심사 의견 등록, 의결 상신, 거래상태 변경 예약, 알림 확인, 추이 비교, 드릴다운 조회, 담당 업무 큐 이동
- 결재/승인: 조회와 단순 필터링은 결재 없이 가능하되, 상태 변경이나 데이터 추출이 발생하면 승인 흐름으로 전환한다.
- 연계 메뉴: 거래지원 심사, 거래상태 관리, 공지 관리, 시장감시, 준법 검토, 감사로그
- 권한/감사: 조회 사유, 검색 조건, 마스킹 해제, 다운로드, 등록/수정/삭제, 승인/반려, 실행 결과와 실패 사유를 관리자 감사로그로 남긴다.
