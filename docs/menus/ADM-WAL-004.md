# ADM-WAL-004 가상자산 입금 현황

## 기본 정보

- 메뉴 코드: ADM-WAL-004
- 상단 메뉴: 입출금·지갑
- 좌측 1메뉴: 입금 운영
- 좌측 2메뉴: 가상자산 입금 현황
- 상태: 정의됨
- 구 어드민 매핑: 신규 정책 정의

## 화면 목적

고객 가상자산 입금의 탐지, 컨펌, 회원 매칭, 반영 상태를 조회한다.

## 업무 설명

입금주소, TXID, 컨펌수, 회원 매칭, 반영금액, 원장 반영 상태를 확인하는 조회 화면이다. 미매칭, 태그 누락, 수동 반영이 필요한 건은 ADM-WAL-005에서 처리한다.

## 결재 프로세스

- 해당사항 없음

## 탭 구성

- 입금 요약
- 컨펌 진행
- 회원 매칭
- 반영 완료
- 미반영 이동

## 탭별 상세 설계

### 입금 요약

- 업무 목적: 입금 처리 현황을 상태별로 조회한다.
- 검색 조건: 기간, 회원번호, 자산, 네트워크, 입금상태, TXID
- 목록 컬럼: 입금탐지시각, 회원번호, 자산, 네트워크, 금액, TXID, 컨펌수, 입금상태, 반영시각
- 상세보기 정보: 입금 탐지 정보, 주소/태그, 컨펌 상태, 회원 매칭, 원장 반영 상태
- 주요 기능: 입금 조회, 상세 이동, TX 확인, 미반영 관리 이동
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-WAL-005 입금 미반영/지연 관리, ADM-WAL-002 입출금 원장조회
- 권한/감사: 입금 조회, TX 링크 이동, 다운로드 이력을 감사로그로 남긴다.

### 컨펌 진행

- 업무 목적: 컨펌 진행 중인 입금 건과 지연 건을 확인한다.
- 검색 조건: 자산, 네트워크, TXID, 현재컨펌수, 필요컨펌수, 지연시간
- 목록 컬럼: 탐지시각, 자산, 네트워크, TXID, 현재컨펌수, 필요컨펌수, 지연시간, 노드상태
- 상세보기 정보: 컨펌 진행률, 노드 응답, 익스플로러 결과, 예상 반영 조건
- 주요 기능: 컨펌 진행 조회, 지연 필터, 익스플로러 이동
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-WAL-003 입출금 이상상태 조회, ADM-WAL-017 지갑잔고 대사
- 권한/감사: 컨펌 조회와 외부 링크 이동 이력을 감사로그로 남긴다.

### 회원 매칭

- 업무 목적: 입금주소/태그와 회원 계정 매칭 상태를 확인한다.
- 검색 조건: 기간, 회원번호, 입금주소, 태그/메모, 매칭상태, 자산
- 목록 컬럼: 탐지시각, 입금주소, 태그/메모, 회원번호, 자산, 금액, 매칭상태, 반영상태
- 상세보기 정보: 주소 등록 정보, 회원 매칭 근거, 태그/메모 일치 여부, 미매칭 사유
- 주요 기능: 회원 매칭 조회, 회원 기본조회 이동, 미매칭 처리 메뉴 이동
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-MEM-001 회원 기본조회, ADM-WAL-005 입금 미반영/지연 관리, ADM-WAL-013 지갑/주소 관리
- 권한/감사: 회원 매칭 조회와 회원 화면 이동 이력을 감사로그로 남긴다.

### 반영 완료

- 업무 목적: 정상 반영된 입금 건과 원장 반영 결과를 확인한다.
- 검색 조건: 기간, 회원번호, 자산, 원장반영상태, TXID
- 목록 컬럼: 반영시각, 회원번호, 자산, 금액, TXID, 입금원장번호, 잔액반영수량, 처리결과
- 상세보기 정보: 입금 원장, 잔액 변동, 정산 연계 상태, 관련 TX
- 주요 기능: 반영 완료 조회, 원장 이동, 회원 이동, 다운로드
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-WAL-002 입출금 원장조회, ADM-MEM-001 회원 기본조회
- 권한/감사: 반영 완료 조회와 다운로드 이력을 감사로그로 남긴다.

### 미반영 이동

- 업무 목적: 미반영 또는 지연 건을 전용 처리 메뉴로 연결한다.
- 검색 조건: 기간, 자산, 미반영사유, 처리필요 여부
- 목록 컬럼: 탐지시각, 자산, 네트워크, TXID, 금액, 미반영사유, 지연시간, 처리메뉴
- 상세보기 정보: 미반영 사유, 처리 필요 기준, 관련 원천 정보
- 주요 기능: 미반영 건 조회, ADM-WAL-005 이동, 링크 복사
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-WAL-005 입금 미반영/지연 관리
- 권한/감사: 미반영 조회와 처리 메뉴 이동 이력을 감사로그로 남긴다.

## 설계 방식 제안

- 이 화면은 입금 현황 조회 화면이며 수동 반영 기능을 포함하지 않는다.
- 미반영/지연 처리는 ADM-WAL-005에서 수행한다.
- 고객 잔고 반영에 영향을 주는 액션은 결재가 있는 처리 메뉴로 분리한다.

## 개발 지침

ADM-WAL-004 화면은 가상자산 입금 상태 조회를 기준으로 개발한다. 수동 반영, 회원 매칭 확정, 원장 보정 기능은 포함하지 않는다.
설계와 다르게 구현해야 하는 항목은 변경 사유와 대체 정책을 작업 기록에 남긴다.
