# ADM-MKT-005 거래량 현황

## 기본 정보

- 메뉴 코드: ADM-MKT-005
- 상단 메뉴: 거래·마켓 관리
- 좌측 1메뉴: 거래 현황
- 좌측 2메뉴: 거래량 현황
- 상태: 정의됨
- 구 어드민 매핑: 모니터링 > 거래 현황 > 거래량

## 탭별 상세 설계

### 운영정책 요약

- 업무 목적: 거래량 현황의 운영정책 요약 탭은 거래 운영자, 시장감시 담당자, 상품 담당자, 준법 담당자가 거래량 분석 업무를 운영정책 요약 관점에서 판단하고 다음 처리 화면으로 자연스럽게 이동하도록 설계한다.
- 검색 조건: 기간, 마켓/페어, 회원번호, 주문번호, 체결번호, 주문상태, 이상거래 유형, 운영정책 요약 상태/구분
- 목록 컬럼: 마켓, 주문/체결 건수, 거래금액, 회원 영향도, 이상징후, 상태, 최근 발생시각, 조치상태, 운영정책 요약 주요 판단값
- 상세보기 정보: 주문/체결 상세, 호가/가격 영향, 관련 회원, 시장경보, 이상거래 근거, 조치 이력, 운영정책 요약별 판단 기준, 운영자 메모, 관련 증빙
- 주요 기능: 상세 조회, 시장감시 Case 생성, 거래제한 요청, 공지/알림 연계, 보고서 추출, 알림 확인, 추이 비교, 드릴다운 조회, 담당 업무 큐 이동, 설정 초안 저장, 영향도 검증, 적용 예약, 비활성 처리, 롤백 요청
- 결재/승인: 등록, 변경, 해제, 제한 적용, 대량 처리, 외부 보고 값 확정은 결재 상신 후 승인 완료 시 실행한다.
- 연계 메뉴: 거래 현황, 회원 기본조회, 시장감시 모니터링, FDS Case, 거래지원 상태, 보고서 센터
- 권한/감사: 조회 사유, 검색 조건, 마스킹 해제, 다운로드, 등록/수정/삭제, 승인/반려, 실행 결과와 실패 사유를 관리자 감사로그로 남긴다.
