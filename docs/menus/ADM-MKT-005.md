# ADM-MKT-005 거래량 현황

## 기본 정보

- 메뉴 코드: ADM-MKT-005
- 상단 메뉴: 거래·마켓 관리
- 좌측 1메뉴: 거래 현황
- 좌측 2메뉴: 거래량 현황
- 상태: 정의됨
- 구 어드민 매핑: 모니터링 > 거래 현황 > 거래량

## 화면 목적

기간별 거래량, 거래대금, 수수료 기여도와 페어별 추이를 분석한다.

## 업무 설명

일/주/月/사용자 지정 기간 기준으로 거래량과 거래대금 추이를 조회하는 분석 화면이다. ADM-MKT-004가 실시간 운영 모니터링이라면, ADM-MKT-005는 기간 비교, 페어별 추이, 급증락 분석, 수수료 기여도를 확인하는 화면으로 역할을 분리한다.

## 결재 프로세스

- 해당사항 없음

## 탭 구성

- 거래량 요약
- 페어별 추이
- 급증락 분석
- 수수료 기여도

## 탭별 상세 설계

### 거래량 요약

- 업무 목적: 선택 기간의 전체 거래량, 거래대금, 체결건수, 활성회원수, 수수료를 요약한다.
- 검색 조건: 기간, 집계단위, 마켓/페어, 기준통화, 거래지원상태
- 목록 컬럼: 기준일, 거래량, 거래대금, 체결건수, 주문건수, 활성회원수, 수수료금액, 전일/전월 대비
- 상세보기 정보: 집계 기준, 기간 합계, 전월/전년 동기 비교, 상위 페어, 데이터 갱신시각
- 주요 기능: 기간 조회, 집계단위 변경, 기간 비교, 페어 상세 이동, 다운로드
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-MKT-004 거래 현황, ADM-MKT-002 체결 내역, ADM-SETTLE-001 정산 대시보드
- 권한/감사: 거래량 조회, 기간 비교, 다운로드 이력을 감사로그로 남긴다.

### 페어별 추이

- 업무 목적: 페어별 거래량과 거래대금 추이를 비교해 시장 흐름과 운영 변화를 확인한다.
- 검색 조건: 기간, 집계단위, 마켓/페어, 거래량 범위, 거래대금 범위
- 목록 컬럼: 마켓, 기준일, 거래량, 거래대금, 체결건수, 활성회원수, 평균체결가, 전일대비, 점유율
- 상세보기 정보: 페어별 시계열 추이, 점유율 변화, 가격 변동과 거래량 상관, 관련 시장경보
- 주요 기능: 페어별 비교, 추이 차트, 상위/하위 페어 정렬, 거래 현황 이동, 다운로드
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-MKT-004 거래 현황, ADM-MKT-002 체결 내역
- 권한/감사: 페어별 조회, 차트 조건, 다운로드 이력을 감사로그로 남긴다.

### 급증락 분석

- 업무 목적: 거래량 또는 거래대금이 급증/급락한 페어를 찾아 원인 확인이 필요한 대상을 식별한다.
- 검색 조건: 기간, 비교 기준, 증감률 기준, 마켓/페어, 최소 거래대금
- 목록 컬럼: 마켓, 기준기간 거래량, 비교기간 거래량, 증감률, 거래대금 증감률, 체결건수 증감률, 활성회원 증감률, 관련 경보건수
- 상세보기 정보: 급증락 산출 기준, 비교 기간, 관련 시장경보, 상위 영향 주문/체결, 영향 회원수
- 주요 기능: 급증락 조건 설정, 대상 정렬, 거래 현황 이동, 관련 체결 조회, 다운로드
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-MKT-004 거래 현황, ADM-MKT-001 거래 요청 내역, ADM-MKT-002 체결 내역
- 권한/감사: 분석 조건, 상세 조회, 다운로드 이력을 감사로그로 남긴다.

### 수수료 기여도

- 업무 목적: 페어별, 기간별 거래 수수료 기여도를 확인해 매출/정산 분석에 활용한다.
- 검색 조건: 기간, 집계단위, 마켓/페어, 수수료 통화, 회원군, 프로모션 적용 여부
- 목록 컬럼: 마켓, 거래대금, 체결건수, 수수료율, 수수료금액, 수수료통화, 프로모션감면액, 순수수료, 기여도
- 상세보기 정보: 수수료 산출 기준, 회원군/멤버십 영향, 프로모션 감면 내역, 정산 반영 상태
- 주요 기능: 수수료 기여도 조회, 페어별 정렬, 정산 화면 이동, 기간 비교, 다운로드
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-MKT-002 체결 내역, ADM-SETTLE-001 정산 대시보드, ADM-SETTLE-006 수수료 정산
- 권한/감사: 수수료 조회, 정산 이동, 다운로드 이력을 감사로그로 남긴다.

## 설계 방식 제안

- 이 화면은 기간 분석 화면으로 설계하고 실시간 모니터링은 ADM-MKT-004에서 수행한다.
- 다운로드는 별도 탭으로 분리하지 않고 각 탭의 목록 기능으로 제공한다.
- 거래량 분석 결과를 근거로 조치가 필요하면 시장경보, FDS, 공지, 보고 등 별도 메뉴로 이동한다.

## 개발 지침

ADM-MKT-005 화면은 거래량/거래대금/수수료의 기간 분석을 기준으로 개발한다. 분석 결과에 따른 직접 조치 기능은 이 메뉴에 포함하지 않는다.
설계와 다르게 구현해야 하는 항목은 변경 사유와 대체 정책을 작업 기록에 남긴다.
