# ADM-DASH-001 운영 대시보드

## 기본 정보

- 메뉴 코드: ADM-DASH-001
- 상단 메뉴: 대시보드
- 좌측 1메뉴: 통합 대시보드
- 좌측 2메뉴: 운영 대시보드
- 상태: 정의됨
- 구 어드민 매핑: 모니터링 모니터링 > 대시보드 

## 화면 목적

거래소 운영 상태를 한 화면에서 확인

## 업무 설명

실시간 거래량, 입출금 건수, 회원가입/KYC 현황, 장애 알림, 지갑 상태, 점검 현황

## 탭 구성

- 운영 요약
- 실시간 지표
- 알림·이벤트
- 처리 현황
- 추이·리포트

## 설계 방식 제안

- 목록, 상세, 처리 이력, 변경 이력을 기준 구조로 둔다.
- 조회 권한과 처리 권한을 분리한다.
- 변경성 액션은 사유 입력과 감사로그를 필수로 둔다.
- 관련 회원, 거래, 입출금, Case, 보고서 화면으로 이동할 수 있게 연결한다.

## 개발 지침

ADM-DASH-001 화면은 이 문서의 화면 목적, 탭 구성, 설계 방식, 권한/감사 기준을 기준으로 개발한다.
설계와 다르게 구현해야 하는 항목은 변경 사유와 대체 정책을 작업 기록에 남긴다.
