# ADM-DASH-001 운영 대시보드

## 기본 정보

- 메뉴 코드: ADM-DASH-001
- 상단 메뉴: 대시보드
- 좌측 1메뉴: 통합 대시보드
- 좌측 2메뉴: 운영 대시보드
- 상태: 정의됨
- 구 어드민 매핑: 모니터링 모니터링 > 대시보드 

## 화면 목적

거래소 운영 상태를 한 화면에서 확인

## 업무 설명

실시간 거래량, 입출금 건수, 회원가입/KYC 현황, 장애 알림, 지갑 상태, 점검 현황

## 탭 구성

- 운영 요약
- 실시간 지표
- 알림·이벤트
- 처리 현황
- 추이·리포트


## 탭별 상세 설계

### 운영 요약

- 업무 목적: 운영 대시보드의 운영 요약 탭은 운영 리더, 승인권자, 준법/리스크 담당자가 거래소 운영 상태를 한 화면에서 확인 업무를 운영 요약 관점에서 판단하고 다음 처리 화면으로 자연스럽게 이동하도록 설계한다.
- 검색 조건: 기간, 업무구분, 상태, 담당부서, SLA 초과 여부, 알림 등급, 운영 요약 상태/구분
- 목록 컬럼: 업무구분, 건수, 미처리 수, SLA 초과, 담당부서, 최근 갱신시각, 바로가기, 운영 요약 주요 판단값
- 상세보기 정보: 지표 산출 기준, 원천 메뉴, 최근 이벤트, 미처리 사유, 담당자 메모, 관련 화면 링크, 운영 요약별 판단 기준, 운영자 메모, 관련 증빙
- 주요 기능: 필터 저장, 담당자 배정, 관련 목록 이동, 보고서 다운로드, 알림 확인, 알림 확인, 추이 비교, 드릴다운 조회, 담당 업무 큐 이동
- 결재/승인: 조회와 단순 필터링은 결재 없이 가능하되, 상태 변경이나 데이터 추출이 발생하면 승인 흐름으로 전환한다.
- 연계 메뉴: 승인 대기, 입출금 처리, FDS/KYT Case, 고객상담, 보고서 센터
- 권한/감사: 조회 사유, 검색 조건, 마스킹 해제, 다운로드, 등록/수정/삭제, 승인/반려, 실행 결과와 실패 사유를 관리자 감사로그로 남긴다.
