# ADM-MEM-013 고액자산가

## 기본 정보

- 메뉴 코드: ADM-MEM-013
- 상단 메뉴: 회원·고객확인
- 좌측 1메뉴: 고객등급/특수고객
- 좌측 2메뉴: 고액자산가
- 상태: 정의됨
- 구 어드민 매핑: 회원관리 회원관리 > 회원관리 > 고액 자산가

## 화면 목적

기준시점별 고객 자산 평가액과 고액자산가 해당 여부를 조회한다.

## 업무 설명

고객별 원화, 가상자산 보유수량, 원화환산 평가금액을 기준으로 고액자산가 해당 여부를 확인하는 조회 화면이다. AML 위험평가, 고객 위험등급 산정, EDD/STR 판단은 AML 시스템에서 수행하며, 어드민은 AML로 제공되는 자산 평가 기준 데이터와 산출 근거를 확인하는 역할로 한정한다.

## 결재 프로세스

- 해당사항 없음

## 탭 구성

- 고액자산가 요약
- 고객별 자산 상세
- 평가·산출 기준
- 변동 이력

## 탭별 상세 설계

### 고액자산가 요약

- 업무 목적: 기준시점별 고액자산가 대상 고객 목록과 평가금액을 조회한다.
- 검색 조건: 기준시점, 회원번호, 고객명/식별값, 총평가금액 범위, 고액자산가 해당 여부, KYC 상태, 계정상태
- 목록 컬럼: 회원번호, 고객명/식별값, 원화 평가금액, 가상자산 평가금액, 총평가금액, 고액자산가 해당 여부, 기준시점, 최근 갱신시각
- 상세보기 정보: 고객 식별정보, 총평가금액 구성, 원화/가상자산 평가금액, 기준금액 충족 여부, 고객별 자산 상세 링크, 회원 기본조회 링크
- 주요 기능: 고액자산가 대상 조회, 평가금액 정렬, 기준시점 변경, 회원 상세 이동, 고객별 자산 상세 이동, 권한 기반 다운로드
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-MEM-001 회원 기본조회, ADM-MKT-003 자산별 보유수량, ADM-SETTLE-014 종가/경제적가치
- 권한/감사: 고객 단위 조회, 마스킹 해제, 다운로드, 상세 이동 이력을 관리자 감사로그로 남긴다.

### 고객별 자산 상세

- 업무 목적: 특정 고객의 자산별 보유수량과 원화환산 평가금액을 확인한다.
- 검색 조건: 회원번호, 기준시점, 자산, 평가금액 범위, 보유수량 범위
- 목록 컬럼: 회원번호, 자산, 티커, 보유수량, 사용가능수량, 주문묶임수량, 출금대기수량, 평가단가, 평가금액, 평가기준일
- 상세보기 정보: 자산별 수량 구성, 원화환산가, 평가금액 산출식, 원천 보유수량 기준, 관련 주문/체결/입출금 링크
- 주요 기능: 고객별 자산 조회, 자산별 평가금액 확인, 보유수량 원천 이동, 회원 기본조회 이동, 권한 기반 다운로드
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-MEM-001 회원 기본조회, ADM-MKT-003 자산별 보유수량, ADM-WAL-001 입출금 조회, ADM-MKT-002 체결 내역
- 권한/감사: 회원 자산 상세 조회, 관련 화면 이동, 다운로드 이력을 감사로그로 남긴다.

### 평가·산출 기준

- 업무 목적: 고액자산가 해당 여부를 산출하는 기준금액, 평가 기준시점, 원화환산가 출처와 산출식을 확인한다.
- 검색 조건: 기준시점, 자산, 평가단가 출처, 적용상태
- 목록 컬럼: 기준시점, 기준금액, 자산, 평가단가, 평가단가 출처, 포함/제외 여부, 적용상태, 갱신시각
- 상세보기 정보: 고액자산가 판단 기준, 총평가금액 산출식, 원화 포함 기준, 가상자산 포함/제외 기준, 평가단가 원천, AML 실시간 API 제공 기준
- 주요 기능: 산출 기준 조회, 평가단가 출처 확인, 포함/제외 자산 확인, 종가/경제적가치 화면 이동
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-SETTLE-014 종가/경제적가치, ADM-MKT-003 자산별 보유수량, ADM-SEC-007 감사로그
- 권한/감사: 산출 기준 조회, 원천 화면 이동, 다운로드 이력을 감사로그로 남긴다.

### 변동 이력

- 업무 목적: 고객이 고액자산가 대상에 진입하거나 이탈한 이력을 기준시점별로 추적한다.
- 검색 조건: 기간, 회원번호, 변동유형, 총평가금액 증감 범위, 기준시점
- 목록 컬럼: 기준일, 회원번호, 변동유형, 이전 총평가금액, 현재 총평가금액, 증감액, 증감률, 이전 대상 여부, 현재 대상 여부
- 상세보기 정보: 변동 원인 후보, 자산별 증감 내역, 평가단가 변경 영향, 보유수량 변경 영향, 관련 주문/체결/입출금 링크
- 주요 기능: 진입/이탈 이력 조회, 금액 증감 정렬, 자산별 변동 상세 확인, 관련 원천 화면 이동, 권한 기반 다운로드
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-MKT-003 자산별 보유수량, ADM-MKT-002 체결 내역, ADM-WAL-001 입출금 조회, ADM-SETTLE-014 종가/경제적가치
- 권한/감사: 변동 이력 조회, 관련 화면 이동, 다운로드 이력을 감사로그로 남긴다.

## AML 연계 기준

- 본 화면의 기준시점별 평가 데이터는 AML 시스템에서 실시간 API로 조회 또는 활용한다.
- AML 전송 현황, 전송 실패, 재전송 관리는 이 메뉴의 업무 범위에 포함하지 않는다.
- AML 위험평가, 고객 위험등급, EDD/STR 판단 및 후속 관리는 AML 시스템에서 수행한다.
- 어드민은 AML 판단 결과를 수정하거나 위험등급을 변경하지 않는다.

## 설계 방식 제안

- 이 화면은 고액자산가 대상 산출 결과와 근거를 확인하는 조회 화면으로 설계한다.
- 고객별 자산 원천은 자산별 보유수량, 체결, 입출금, 종가/경제적가치 화면으로 이동해 확인한다.
- 기준금액이나 산출 조건을 변경하는 기능은 이 메뉴에 두지 않는다. 정책값 변경이 필요하면 별도 설정 메뉴에서 결재 프로세스를 태운다.
- 다운로드는 별도 탭으로 분리하지 않고 각 목록의 권한 기반 기능으로 제공한다.

## 개발 지침

ADM-MEM-013 화면은 기준시점별 고객 자산 평가액과 고액자산가 해당 여부 조회를 기준으로 개발한다. AML 위험판단, 전송 관리, 고객 제한, 수동 등급 변경 기능은 이 메뉴에 포함하지 않는다.
설계와 다르게 구현해야 하는 항목은 변경 사유와 대체 정책을 작업 기록에 남긴다.
