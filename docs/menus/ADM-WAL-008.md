# ADM-WAL-008 출금 승인/보류

## 기본 정보

- 메뉴 코드: ADM-WAL-008
- 상단 메뉴: 입출금·지갑
- 좌측 1메뉴: 출금 운영
- 좌측 2메뉴: 출금 승인/보류
- 상태: 준비중
- 구 어드민 매핑: 신규 정책 정의

## 화면 목적

고위험 또는 수동 확인이 필요한 출금의 승인, 반려, 보류 해제를 처리한다.

## 업무 설명

고액 출금, 의심거래, KYT 위험, Travel Rule 오류, 제재 위험, 계정 위험 등으로 자동 처리되지 않은 출금 건을 검토하고 승인/반려/보류 유지/보류 해제를 수행하는 처리 화면이다.

## 결재 프로세스

- 승인·반려: 출금 승인, 반려, 보류 해제, 예외 승인은 외부 전송에 영향을 주므로 승인 완료 후 실행한다.

## 탭 구성

- 승인 대기
- 위험 근거
- Travel Rule·KYT
- 승인·반려
- 처리 이력

## 탭별 상세 설계

### 승인 대기

- 업무 목적: 수동 승인 또는 보류 해제가 필요한 출금 대상 목록을 확인한다.
- 검색 조건: 기간, 회원번호, 자산, 금액, 위험유형, 승인상태, 보류사유
- 목록 컬럼: 신청시각, 회원번호, 자산, 출금금액, 위험유형, 보류사유, 승인상태, 담당자, SLA
- 상세보기 정보: 출금 신청 정보, 회원 상태, 주소, 위험 신호, 승인 필요 사유
- 주요 기능: 승인 대상 조회, 담당자 배정, 상세 검토 이동
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-WAL-007 가상자산 출금 현황, ADM-MEM-001 회원 기본조회
- 권한/감사: 승인 대기 조회와 담당자 배정 이력을 감사로그로 남긴다.

### 위험 근거

- 업무 목적: 출금 보류 또는 승인 필요 판단의 근거를 확인한다.
- 검색 조건: 위험유형, 위험등급, 회원번호, 자산, 탐지원천
- 목록 컬럼: 출금번호, 회원번호, 위험유형, 위험등급, 탐지원천, 탐지시각, 근거요약
- 상세보기 정보: FDS/KYT/계정위험/제재 결과, 최근 활동, 관련 상담/차단 이력
- 주요 기능: 위험 근거 조회, 관련 위험 화면 이동, 검토 의견 등록
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-RISK-011 KYT Alert, ADM-RISK-008 FDS 해제 승인, ADM-MEM-004 로그인·보안 조회
- 권한/감사: 위험 근거 조회와 의견 등록 이력을 감사로그로 남긴다.

### Travel Rule·KYT

- 업무 목적: Travel Rule 송수신 결과와 KYT 위험 평가를 출금 승인 판단에 반영한다.
- 검색 조건: 기간, Travel Rule 상태, VASP, KYT 위험등급, 회원번호
- 목록 컬럼: 출금번호, VASP, 수신자정보상태, Travel Rule 상태, KYT 위험등급, 제재위험, 처리필요 여부
- 상세보기 정보: Travel Rule 전문 상태, VASP 검증, 주소 위험도, KYT 근거, 보완 필요 사항
- 주요 기능: Travel Rule/KYT 결과 조회, 관련 VASP/KYT 화면 이동, 보완 메모
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-RISK-015 트래블룰 송수신 이력, ADM-RISK-011 KYT Alert, ADM-RISK-013 KYT VASP 차단 관리
- 권한/감사: Travel Rule/KYT 조회와 메모 이력을 감사로그로 남긴다.

### 승인·반려

- 업무 목적: 검토 결과에 따라 출금 승인, 반려, 보류 유지, 보류 해제를 처리한다.
- 검색 조건: 기간, 승인상태, 승인자, 회원번호, 자산, 결과
- 목록 컬럼: 상신번호, 출금번호, 회원번호, 자산, 금액, 승인상태, 승인자, 실행상태, 처리시각
- 상세보기 정보: 상신 내용, 검토 근거, 승인/반려 의견, 변경 전후 상태, 실행 결과, 실패 사유
- 주요 기능: 승인, 반려, 보완 요청, 보류 해제, 승인 결과 실행
- 결재/승인: 출금 승인, 반려, 보류 해제, 예외 승인은 외부 전송에 영향을 주므로 승인 완료 후 실행한다.
- 연계 메뉴: ADM-WAL-007 가상자산 출금 현황, ADM-WAL-009 출금 재처리/실패관리, ADM-SEC-007 감사로그
- 권한/감사: 승인/반려, 보류 해제, 실행 결과를 감사로그로 남긴다.

### 처리 이력

- 업무 목적: 출금 승인/보류 처리 결과와 감사 근거를 조회한다.
- 검색 조건: 기간, 회원번호, 자산, 처리결과, 승인자, 담당자
- 목록 컬럼: 처리시각, 출금번호, 회원번호, 자산, 금액, 처리결과, 승인자, 사유
- 상세보기 정보: 처리 타임라인, 승인 의견, 실행 결과, 관련 출금 현황, 감사로그
- 주요 기능: 처리 이력 조회, 출금 현황 이동, 감사로그 이동
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-WAL-007 가상자산 출금 현황, ADM-SEC-007 감사로그
- 권한/감사: 처리 이력 조회와 다운로드 이력을 감사로그로 남긴다.

## 설계 방식 제안

- 승인 판단 근거와 승인 실행을 분리하되 같은 업무 흐름 안에서 추적 가능하게 한다.
- 승인 없는 보류 해제나 예외 출금 처리는 허용하지 않는다.
- KYT, Travel Rule, FDS 근거는 원천 화면과 연결한다.

## 개발 지침

ADM-WAL-008 화면은 고위험 출금의 승인/반려/보류 해제 결재 흐름을 기준으로 개발한다. 승인 완료 전 외부 전송이 실행되지 않도록 한다.
설계와 다르게 구현해야 하는 항목은 변경 사유와 대체 정책을 작업 기록에 남긴다.
