# ADM-MKT-011 멤버십 조회

## 기본 정보

- 메뉴 코드: ADM-MKT-011
- 상단 메뉴: 거래·마켓 관리
- 좌측 1메뉴: 수수료·등급 정책
- 좌측 2메뉴: 멤버십 조회
- 상태: 정의됨
- 구 어드민 매핑: 기존 거래관리 > 멤버십 조회

## 화면 목적

회원 등급과 수수료 적용 상태 조회

## 업무 설명

거래량 기준, 등급, 적용수수료, 산정기간, 예외수수료

## 탭 구성

- 등급현황
- 산정기준
- 변경이력
- 예외

## 설계 방식 제안

- 목록, 상세, 처리 이력, 변경 이력을 기준 구조로 둔다.
- 조회 권한과 처리 권한을 분리한다.
- 변경성 액션은 사유 입력과 감사로그를 필수로 둔다.
- 관련 회원, 거래, 입출금, Case, 보고서 화면으로 이동할 수 있게 연결한다.

## 개발 지침

ADM-MKT-011 화면은 이 문서의 화면 목적, 탭 구성, 설계 방식, 권한/감사 기준을 기준으로 개발한다.
설계와 다르게 구현해야 하는 항목은 변경 사유와 대체 정책을 작업 기록에 남긴다.
