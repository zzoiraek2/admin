- 종료심사 사유 접수: 종료심사 개시는 대외 공지 연계 가능성이 있으므로 승인자 확인과 사유 기록을 요구한다.
- 종료 영향 분석: 거래종료 일정 확정, 입출금 중단, 고객 공지는 결재 완료 후 적용한다.

- 리스크 검토: 고위험 또는 예외 승인 필요 항목은 의결·승인 탭에서 승인 흐름으로 확정한다.
- 의결·승인: 의결·승인 결과는 승인 완료 전까지 거래상태와 대외 공지에 반영하지 않는다.
