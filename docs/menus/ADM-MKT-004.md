# ADM-MKT-004 거래 현황

## 기본 정보

- 메뉴 코드: ADM-MKT-004
- 상단 메뉴: 거래·마켓 관리
- 좌측 1메뉴: 거래 현황
- 좌측 2메뉴: 거래 현황
- 상태: 정의됨
- 구 어드민 매핑: 모니터링 > 거래 현황

## 화면 목적

전체 거래 상태와 페어별 시장경보 발생 현황을 실시간 운영 관점에서 모니터링한다.

## 업무 설명

거래대금, 체결건수, 주문건수, 취소율, 활성회원, 페어별 가격/거래량/스프레드, 시장경보 알림 발생 현황을 확인하는 운영 모니터링 화면이다. 주문장 원장과 체결 원장을 중복 제공하지 않고, 이상 징후를 발견하면 관련 주문/체결/회원/FDS 화면으로 이동하는 구조로 설계한다.

## 결재 프로세스

- 해당사항 없음

## 탭 구성

- 거래 요약
- 실시간 주문·체결
- 페어별 현황
- 시장경보 모니터링

## 탭별 상세 설계

### 거래 요약

- 업무 목적: 전체 거래 상태를 KPI로 확인해 운영 이상 여부를 빠르게 판단한다.
- 검색 조건: 기준일, 집계구간, 마켓/페어, 거래지원상태
- 목록 컬럼: 마켓, 거래대금, 체결건수, 주문건수, 취소건수, 취소율, 활성회원수, 전일대비, 최근 갱신시각
- 상세보기 정보: KPI 산출 기준, 전일/전주 대비 추이, 상위/하위 페어, 최근 경보 발생 요약
- 주요 기능: 거래 KPI 조회, 집계구간 변경, 페어 상세 이동, 관련 원장 화면 이동
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-MKT-001 거래 요청 내역, ADM-MKT-002 체결 내역, ADM-MKT-005 거래량 현황
- 권한/감사: 모니터링 조회와 상세 이동 이력을 감사로그로 남긴다.

### 실시간 주문·체결

- 업무 목적: 현재 주문과 체결 흐름이 정상적으로 발생하는지 실시간 지표로 확인한다.
- 검색 조건: 마켓/페어, 시간 범위, 주문상태, 체결 발생 여부, 지연 기준
- 목록 컬럼: 마켓, 최근주문시각, 최근체결시각, 대기주문수, 부분체결수, 체결건수, 평균체결간격, 지연여부
- 상세보기 정보: 최근 주문/체결 흐름, 지연 구간, 주문장 링크, 체결 원장 링크, 시스템 상태 참고 링크
- 주요 기능: 실시간 새로고침, 지연 페어 필터, 주문장 이동, 체결 내역 이동
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-MKT-001 거래 요청 내역, ADM-MKT-002 체결 내역, ADM-DASH-001 운영 대시보드
- 권한/감사: 실시간 조회와 관련 화면 이동 이력을 감사로그로 남긴다.

### 페어별 현황

- 업무 목적: 페어 단위 가격, 거래량, 스프레드, 체결 상태를 비교해 운영 상태를 확인한다.
- 검색 조건: 마켓/페어, 거래지원상태, 가격변동률 범위, 거래량변동률 범위, 스프레드 범위
- 목록 컬럼: 마켓, 현재가, 전일대비, 거래량, 거래대금, 스프레드, 체결건수, 주문건수, 활성회원수, 경보건수
- 상세보기 정보: 페어별 가격/거래량 추이, 호가 상태, 주문/체결 연결, 관련 시장경보, 영향 회원수
- 주요 기능: 페어 비교, 정렬/필터, 거래량 현황 이동, 주문/체결 원장 이동
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-MKT-005 거래량 현황, ADM-MKT-001 거래 요청 내역, ADM-MKT-002 체결 내역
- 권한/감사: 페어별 조회, 정렬/필터, 다운로드 이력을 감사로그로 남긴다.

### 시장경보 모니터링

- 업무 목적: 페어별 시장경보 알림 발생 현황을 확인하고 관련 원장으로 이동해 이상 거래 흐름을 점검한다.
- 검색 조건: 기간, 마켓/페어, 경보유형, 심각도, 발생상태
- 목록 컬럼: 발생시각, 마켓/페어, 경보유형, 심각도, 기준가격, 기준거래량, 영향 회원수, 반복발생여부, 확인상태
- 상세보기 정보: 경보 발생 근거, 가격/거래량 변동, 연관 주문/체결, 영향 회원 목록, 반복 발생 여부, 확인 메모
- 주요 기능: 경보 상세 조회, 관련 주문/체결/회원 이동, 확인 메모 등록, 기간별 경보 필터링
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-MKT-001 거래 요청 내역, ADM-MKT-002 체결 내역, ADM-MEM-001 회원 기본조회, ADM-RISK-008 FDS 해제 승인
- 권한/감사: 경보 조회, 관련 주문/회원 이동, 확인 메모, 다운로드 이력을 감사로그로 남긴다.

## 설계 방식 제안

- 이 화면은 모니터링 화면으로 설계하고 주문장/체결 원장의 상세 원본은 ADM-MKT-001, ADM-MKT-002로 이동해 확인한다.
- `회원 영향도`는 별도 탭으로 두지 않고 페어별 현황 또는 시장경보 상세의 컬럼과 상세 정보로 제공한다.
- 시장감시 사건 목록이나 처리 상태 관리는 이 메뉴에 두지 않는다. 조치가 필요한 경우 FDS, 제한, 공지, 보고 등 별도 업무 메뉴로 이동한다.

## 개발 지침

ADM-MKT-004 화면은 거래 운영 모니터링과 시장경보 조회를 기준으로 개발한다. 거래 제한, 공지 확정, 보고 확정 같은 후속 조치 기능은 이 메뉴에 포함하지 않는다.
설계와 다르게 구현해야 하는 항목은 변경 사유와 대체 정책을 작업 기록에 남긴다.
