## 설계 방식 제안

- 이 화면은 실시간 지갑잔고 대사 모니터링 화면으로 설계한다.
- 보정, 회계 마감, 일일 정산 확정 기능은 포함하지 않는다.
- 차이 발생 시 원인 후보와 관련 원천 화면으로 이동할 수 있게 하고, 실제 정산/보정 처리는 정산·회계 메뉴에서 수행한다.
- 다운로드는 별도 탭으로 분리하지 않고 목록 권한 기능으로 제공한다.

## 개발 지침
