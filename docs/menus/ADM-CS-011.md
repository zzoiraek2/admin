# ADM-CS-011 팝업 관리

## 기본 정보

- 메뉴 코드: ADM-CS-011
- 상단 메뉴: 고객지원·서비스
- 좌측 1메뉴: 화면관리
- 좌측 2메뉴: 팝업 관리
- 상태: 정의됨
- 구 어드민 매핑: 시스템 관리 시스템 관리 > 거래소 화면 관리 > 팝업 관리 

## 화면 목적

팝업 공지 관리

## 업무 설명

팝업내용, 노출조건, 기간, 버튼, 링크

## 탭 구성

- 긴급
- 일반
- 예약
- 종료
- 이력

## 설계 방식 제안

- 목록, 상세, 처리 이력, 변경 이력을 기준 구조로 둔다.
- 조회 권한과 처리 권한을 분리한다.
- 변경성 액션은 사유 입력과 감사로그를 필수로 둔다.
- 관련 회원, 거래, 입출금, Case, 보고서 화면으로 이동할 수 있게 연결한다.

## 개발 지침

ADM-CS-011 화면은 이 문서의 화면 목적, 탭 구성, 설계 방식, 권한/감사 기준을 기준으로 개발한다.
설계와 다르게 구현해야 하는 항목은 변경 사유와 대체 정책을 작업 기록에 남긴다.
