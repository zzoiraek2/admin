## 화면 목적

## 업무 설명

입금 가능 여부, 출금 가능 여부, 자동/수동 처리 모드, 점검 안내 문구, 공지 연계 상태를 관리하는 운영 설정 화면이다. 서비스 중지/재개와 처리 모드 변경은 고객 서비스에 직접 영향을 주므로 결재 후 반영한다.

- 점검·공지 문구: 고객 노출 문구 적용과 공지 연계는 승인 완료 후 반영한다.
