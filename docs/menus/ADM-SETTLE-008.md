## 화면 목적

회사 보유수량, 평가금액, 지갑별 잔고, 변동내역


## 결재 프로세스

- 평가 요약: 조회와 필터링은 결재 없이 가능하지만 상태 변경이나 다운로드는 권한과 사유를 확인한다.
- 기준가·예외: 정책 변경은 승인 완료 후 예약 시점에 반영하고 변경 전후 값을 반드시 보관한다.
- 수동 보정: 조회와 맥락 확인은 결재 없이 가능하지만 상태 변경이나 외부 영향 조치는 승인 흐름으로 전환한다.
- 승인·반영: 승인 완료 전에는 외부 영향이 있는 상태 변경, 차단, 지급, 게시, 보고값 확정을 실행하지 않는다.
