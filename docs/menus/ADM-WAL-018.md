## 화면 목적

고객자산 원장 합계와 실제 보관자산의 실시간 차이를 모니터링한다.

## 업무 설명

자산별 고객 원장 합계, 실제 지갑/보관 수량, 회사자산, 수수료분, 보관방식별 수량을 비교해 고객자산 보관 상태를 실시간으로 확인하는 화면이다. 일일 정산과 회계 마감은 정산·회계 영역에서 수행하며, 이 메뉴는 부족 여부와 차이 원인 후보를 조회하는 모니터링 화면으로 한정한다.

## 탭별 상세 설계

- 업무 목적: 자산별 고객 원장 합계와 실제 보관자산을 비교해 부족 또는 초과 상태를 빠르게 확인한다.
- 검색 조건: 기준시각, 자산, 네트워크, 대사상태, 차이수량 범위, 부족 여부
- 목록 컬럼: 자산, 네트워크, 고객원장합계, 실제보관수량, 차이수량, 차이율, 부족여부, 회사자산수량, 수수료분수량, 최근기준시각
- 상세보기 정보: 고객 원장 합계 산출 기준, 보관수량 구성, 회사자산/고객자산 분리 상태, 차이 원인 후보, 관련 지갑잔고 대사 링크
- 주요 기능: 고객자산 대사 조회, 부족 대상 필터링, 자산별 상세 이동, 지갑잔고 대사 이동, 권한 기반 다운로드
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-WAL-017 지갑잔고 대사, ADM-MKT-003 자산별 보유수량, ADM-WAL-015 핫/콜드 비율 관리
- 권한/감사: 대사 현황 조회, 부족 대상 필터, 다운로드, 관련 화면 이동 이력을 감사로그로 남긴다.

- 검색 조건: 자산, 네트워크, 보관방식, 고객/회사 구분, 부족 여부, 기준시각
- 목록 컬럼: 자산, 네트워크, 고객자산수량, 회사자산수량, 수수료분수량, 핫월렛수량, 콜드월렛수량, 위탁수량, 기타보관수량, 차이수량
- 상세보기 정보: 보관방식별 구성, 고객자산/회사자산 분리 기준, 지갑주소별 구성, 위탁기관 정보, 관련 비율 관리 링크
- 주요 기능: 자산별 보관 비교, 보관방식별 필터, 지갑주소 상세 이동, 핫/콜드 비율 관리 이동
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-WAL-013 지갑/주소 관리, ADM-WAL-015 핫/콜드 비율 관리, ADM-WAL-017 지갑잔고 대사
- 권한/감사: 자산별 보관 비교 조회, 주소 상세 이동, 다운로드 이력을 감사로그로 남긴다.

### 고객 원장 합계

- 업무 목적: 고객별 잔고를 직접 관리하지 않고 자산별 고객 원장 합계와 산출 기준을 확인한다.
- 검색 조건: 기준시각, 자산, 원장구분, 집계상태, 차이 여부
- 목록 컬럼: 자산, 고객수, 사용가능합계, 주문묶임합계, 출금대기합계, 제한수량합계, 총고객원장합계, 집계상태, 기준시각

- 주요 기능: 고객 원장 합계 조회, 구성 수량 확인, 자산별 보유수량 이동, 집계상태 필터링

## 설계 방식 제안

- 이 화면은 실시간 고객자산 대사 모니터링 화면으로 설계한다.
- 회사자산 분리보관 점검은 별도 메뉴로 두지 않고 `자산별 보관 비교`에서 고객분/회사분/수수료분을 함께 확인한다.
- 일일 정산, 월말 확정, 회계 마감, 원장 보정은 정산·회계 영역에서 수행한다.
- 다운로드는 별도 탭으로 분리하지 않고 각 목록의 권한 기반 기능으로 제공한다.

## 개발 지침

ADM-WAL-018 화면은 고객 원장 합계와 실제 보관자산의 실시간 비교 및 차이 원인 추적을 기준으로 개발한다. 보정, 회계 마감, 고객자산 확정 보고 기능은 이 메뉴에 포함하지 않는다.
설계와 다르게 구현해야 하는 항목은 변경 사유와 대체 정책을 작업 기록에 남긴다.
