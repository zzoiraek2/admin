# ADM-MEM-010 강제재이행/반려 대상관리

## 기본 정보

- 메뉴 코드: ADM-MEM-010
- 상단 메뉴: 회원·고객확인
- 좌측 1메뉴: 고객확인 관리
- 좌측 2메뉴: 강제재이행/반려 대상관리
- 상태: 정의됨
- 구 어드민 매핑: 회원관리 회원관리 > 고객확인 > 강제재이행/반려 대상관리

### 대상 요약

- 업무 목적: 강제재이행/반려 대상관리의 대상 요약 탭은 CS, KYC 담당자, 준법/FDS 담당자, 승인권자가 특정 회원에게 강제 KYC 재이행 또는 반려 제한 적용 업무를 대상 요약 관점에서 판단하고 다음 처리 화면으로 자연스럽게 이동하도록 설계한다.
- 검색 조건: 회원번호, UID, 이름, 이메일, 휴대폰, CI/DI, KYC 상태, 계정상태, 위험등급, 대상 요약 상태/구분
- 목록 컬럼: 회원번호, 이름/식별값, KYC 상태, 계정상태, 거래/입출금 제한, 위험등급, 최근 활동, 담당자, 대상 요약 주요 판단값
- 상세보기 정보: 회원 식별정보, 연락처, KYC 이력, 계정 보안상태, 제한/차단 상태, 상담/처리 이력, 감사로그, 대상 요약별 판단 기준, 운영자 메모, 관련 증빙
- 주요 기능: 회원 360 조회, 관련 입출금/거래 이동, 상태 변경 요청, 제한/해제 상신, 상담 메모 등록, 알림 확인, 추이 비교, 드릴다운 조회, 담당 업무 큐 이동, 상신, 승인, 반려, 보완 요청, 승인선 변경, 승인 결과 반영, 설정 초안 저장, 영향도 검증, 적용 예약, 비활성 처리, 롤백 요청
- 결재/승인: 등록, 변경, 해제, 제한 적용, 대량 처리, 외부 보고 값 확정은 결재 상신 후 승인 완료 시 실행한다.
- 연계 메뉴: 회원 기본조회, KYC 심사 대기, 고객확인 현황조회, 로그인·보안 조회, 입출금 조회, 차단 등록/해제, 상담 이력
- 권한/감사: 조회 사유, 검색 조건, 마스킹 해제, 다운로드, 등록/수정/삭제, 승인/반려, 실행 결과와 실패 사유를 관리자 감사로그로 남긴다.
