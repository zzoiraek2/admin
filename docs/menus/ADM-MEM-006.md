- 주요 기능: 회원 360 조회, 관련 입출금/거래 이동, 상태 변경 요청, 제한/해제 상신, 상담 메모 등록, 알림 확인, 추이 비교, 드릴다운 조회, 담당 업무 큐 이동, 변경 전후 비교, 감사 로그 다운로드, 관련 승인 건 역추적
