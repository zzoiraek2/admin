# ADM-WAL-012 코인별 입출금 상태

## 기본 정보

- 메뉴 코드: ADM-WAL-012
- 상단 메뉴: 입출금·지갑
- 좌측 1메뉴: 입출금 정책
- 좌측 2메뉴: 코인별 입출금 상태
- 상태: 준비중
- 구 어드민 매핑: 신규 정책 정의

## 화면 목적

자산별 입금/출금 가능 상태, 컨펌수, 수수료, 점검 사유를 관리한다.

## 업무 설명

코인별 입금 가능 여부, 출금 가능 여부, 네트워크 상태, 필요 컨펌수, 출금 수수료, 점검 사유, 고객 노출 상태를 관리하는 정책 화면이다. 자산별 입출금 중지/재개와 수수료/컨펌수 변경은 고객 거래와 입출금에 직접 영향을 주므로 결재 후 반영한다.

## 결재 프로세스

- 입금 정책: 자산별 입금 중지/재개, 필요 컨펌수 변경은 승인 완료 후 반영한다.
- 출금 정책: 자산별 출금 중지/재개, 출금 수수료 변경은 승인 완료 후 반영한다.
- 컨펌·수수료: 컨펌수와 수수료 기준 변경은 승인 완료 후 반영한다.

## 탭 구성

- 출금 정책

- 코인별 상태 조회와 정책 변경을 분리한다.

- 고객 노출 문구와 공지는 고객지원·서비스 공지 메뉴와 연계한다.

## 개발 지침

ADM-WAL-012 화면은 자산별 입출금 상태 조회와 결재 기반 정책 변경을 기준으로 개발한다. 승인 없는 입출금 중지/재개 또는 수수료 변경은 허용하지 않는다.
설계와 다르게 구현해야 하는 항목은 변경 사유와 대체 정책을 작업 기록에 남긴다.
