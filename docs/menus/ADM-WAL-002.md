# ADM-WAL-002 입출금 원장조회

## 기본 정보

- 메뉴 코드: ADM-WAL-002
- 상단 메뉴: 입출금·지갑
- 좌측 1메뉴: 입출금 통합조회
- 좌측 2메뉴: 입출금 원장조회
- 상태: 준비중
- 구 어드민 매핑: 신규 정책 정의

## 화면 목적

회계·정산 기준의 입출금 원장과 잔액 변동 내역을 조회한다.

## 업무 설명

입출금 원장번호, 회원, 자산, 금액, 수수료, 잔액 변동, 회계 반영, 정산 반영 여부를 조회하는 화면이다. 원장 보정이나 회계 마감은 정산·회계 영역에서 수행하고, 이 화면은 원장 조회와 정산 연계 상태 확인으로 한정한다.

## 결재 프로세스

- 해당사항 없음

## 탭 구성

- 원장 요약
- 입금 원장
- 출금 원장
- 잔액 변동
- 정산 연계 상태

## 탭별 상세 설계

### 원장 요약

- 업무 목적: 기간별 입출금 원장 집계와 정산 반영 상태를 확인한다.
- 검색 조건: 기간, 자산, 회원번호, 원장구분, 정산반영상태, 회계반영상태
- 목록 컬럼: 기준일, 입금건수, 출금건수, 입금금액, 출금금액, 수수료, 정산반영건수, 미반영건수
- 상세보기 정보: 원장 집계 기준, 정산 미반영 사유, 회계 반영 상태, 관련 정산 링크
- 주요 기능: 원장 집계 조회, 미반영 필터링, 정산 화면 이동, 다운로드
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-SETTLE-001 정산 대시보드, ADM-WAL-001 입출금 통합현황
- 권한/감사: 원장 조회, 다운로드, 정산 화면 이동 이력을 감사로그로 남긴다.

### 입금 원장

- 업무 목적: 입금 원장 단위로 고객 잔액 증가와 회계/정산 반영 상태를 확인한다.
- 검색 조건: 기간, 회원번호, 자산, 네트워크, 입금원장번호, TXID, 정산반영상태
- 목록 컬럼: 입금원장번호, 회원번호, 자산, 금액, 수수료, TXID, 잔액증가액, 원장상태, 정산반영상태, 생성시각
- 상세보기 정보: 입금 원장 상세, 잔액 반영값, TX 정보, 정산 반영 결과, 관련 입금 현황
- 주요 기능: 입금 원장 조회, TX/입금 현황 이동, 정산 상태 확인
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-WAL-004 가상자산 입금 현황, ADM-SETTLE-001 정산 대시보드
- 권한/감사: 입금 원장 조회와 상세 이동 이력을 감사로그로 남긴다.

### 출금 원장

- 업무 목적: 출금 원장 단위로 고객 잔액 감소와 회계/정산 반영 상태를 확인한다.
- 검색 조건: 기간, 회원번호, 자산, 네트워크, 출금원장번호, TXID, 정산반영상태
- 목록 컬럼: 출금원장번호, 회원번호, 자산, 출금금액, 수수료, TXID, 잔액감소액, 원장상태, 정산반영상태, 생성시각
- 상세보기 정보: 출금 원장 상세, 잔액 반영값, TX 정보, 정산 반영 결과, 관련 출금 현황
- 주요 기능: 출금 원장 조회, TX/출금 현황 이동, 정산 상태 확인
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-WAL-007 가상자산 출금 현황, ADM-SETTLE-001 정산 대시보드
- 권한/감사: 출금 원장 조회와 상세 이동 이력을 감사로그로 남긴다.

### 잔액 변동

- 업무 목적: 입출금 원장이 회원 잔액에 반영된 변동 내역을 추적한다.
- 검색 조건: 기간, 회원번호, 자산, 변동유형, 원장번호
- 목록 컬럼: 변동시각, 회원번호, 자산, 변동유형, 이전잔액, 변동수량, 이후잔액, 원장번호, 원천업무
- 상세보기 정보: 잔액 변동 산출식, 원천 원장, 이전/이후 값, 관련 입출금/정산 링크
- 주요 기능: 잔액 변동 조회, 원장 이동, 회원 기본조회 이동, 다운로드
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-MEM-001 회원 기본조회, ADM-MKT-003 자산별 보유수량
- 권한/감사: 회원 잔액 변동 조회와 다운로드 이력을 감사로그로 남긴다.

### 정산 연계 상태

- 업무 목적: 입출금 원장이 정산·회계로 정상 연계되었는지 확인한다.
- 검색 조건: 기간, 원장구분, 정산반영상태, 회계반영상태, 오류사유
- 목록 컬럼: 원장번호, 원장구분, 자산, 금액, 정산반영상태, 회계반영상태, 오류사유, 최근반영시각
- 상세보기 정보: 정산 연계 요청값, 반영 결과, 오류 로그, 정산 화면 링크
- 주요 기능: 정산 연계 상태 조회, 오류 건 필터링, 정산 화면 이동
- 결재/승인: 해당사항 없음
- 연계 메뉴: ADM-SETTLE-001 정산 대시보드, ADM-SETTLE-002 정산 원장
- 권한/감사: 정산 연계 조회와 관련 화면 이동 이력을 감사로그로 남긴다.

## 설계 방식 제안

- 원장조회는 조회 전용으로 설계하고 보정, 마감, 재처리 기능을 두지 않는다.
- 원장 보정과 회계 마감은 정산·회계 메뉴에서 처리한다.
- 회원 잔액 조회는 권한과 감사로그를 강화한다.

## 개발 지침

ADM-WAL-002 화면은 입출금 원장과 정산 연계 상태 조회를 기준으로 개발한다. 원장 수정, 보정, 회계 마감 확정 기능은 포함하지 않는다.
설계와 다르게 구현해야 하는 항목은 변경 사유와 대체 정책을 작업 기록에 남긴다.
