- 메뉴 코드: ADM-MEM-003
- 상단 메뉴: 회원·고객확인

- 주요 기능: 회원 360 조회, 관련 입출금/거래 이동, 상태 변경 요청, 제한/해제 상신, 상담 메모 등록, 필터 저장, 상세 화면 이동, 담당자 큐 이동, 보고서 다운로드

## 설계 방식 제안

- 목록, 상세, 처리 이력, 변경 이력을 기준 구조로 둔다.
- 조회 권한과 처리 권한을 분리한다.
- 변경성 액션은 사유 입력과 감사로그를 필수로 둔다.
- 관련 회원, 거래, 입출금, Case, 보고서 화면으로 이동할 수 있게 연결한다.
