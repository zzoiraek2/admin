# ADM-WAL-003 입출금 이상상태 조회

## 기본 정보

- 메뉴 코드: ADM-WAL-003
- 상단 메뉴: 입출금·지갑
- 좌측 1메뉴: 입출금 통합조회
- 좌측 2메뉴: 입출금 이상상태 조회
- 상태: 준비중
- 구 어드민 매핑: 신규 정책 정의

## 화면 목적

정상 입출금 흐름에서 벗어난 이상상태를 탐지하고 관련 처리 메뉴로 연결한다.

## 업무 설명

상태 불일치, TXID 누락, 컨펌 지연, 장기 미처리, 지갑 반영 오류, 원장 반영 지연 같은 이상상태를 조회하는 화면이다. 직접 수동반영이나 재처리를 수행하지 않고, 원인 확인 후 전용 처리 메뉴로 이동하는 관제 화면으로 설계한다.
