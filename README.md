# Admin 설계정책서

`admin.csv`에 정리된 어드민 메뉴 정책을 웹 UI로 탐색하는 정적 문서 프로젝트입니다.

## 구성

- `index.html`: 화면 골격
- `styles.css`: 어드민 정책서 UI 스타일
- `app.js`: CSV 파싱, 메뉴 트리, 검색/필터, 설계 의견 생성
- `admin.csv`: 메뉴 정책 원본 데이터
- `vendor/lucide.min.js`: 아이콘 렌더링

## 실행

로컬에서 CSV fetch가 필요하므로 간단한 웹 서버로 실행합니다.

```bash
python -m http.server 4173 --bind 127.0.0.1
```

브라우저에서 `http://127.0.0.1:4173/`을 열면 됩니다.

## 주요 기능

- 상단 메뉴, 좌측 메뉴, 화면 단위 트리 탐색
- 준비중/정의됨 상태 필터
- 메뉴명, 목적, 설명, 구 어드민 매핑, 설계 의견 통합 검색
- 메뉴별 개발 참조 코드 검색과 지침 복사
- 메뉴별 화면 목적, 탭 구성, 구 어드민 매핑 확인
- 메뉴 성격에 따른 설계 방식 제안과 정책 체크리스트 자동 표시
